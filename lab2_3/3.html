Пример Web-документа с применением CSS

Этот документ создан с применением трех методов подключения CSS: inline, внутренний и внешний.

Мы можем использовать различные методы применения CSS в зависимости от требований. Для небольших элементов часто применяются in-line стили, для крупных — внешние таблицы стилей.
